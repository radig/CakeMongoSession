h1. Plugin CakeMongoSession

h2. Descrição

Plugin que visa oferecer uma forma distribuída de armazenamento de sessões dos
usuários, unindo a perfomance e escalabilidade do bando de dados não-relacional MongoDB
com as facilidades do framework CakePHP.

h2. ATENÇÃO

O código não encontra-se funcional.

Da forma como está ele entra em conflito com os componentes internos do CakePHP como o 
AuthComponent.
